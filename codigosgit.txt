git clone -> traz o repositório da núvem para o diretório local
git status -> ciclo de estados dos meus arquivos tracked = verde ou ontracked = vermelho
git add * -> mostrar ao git quais arquivos 
git nome do arquiv -> você envia so ele
touch + nome do arquivo + ponto + tipo, vc cria um arquivo de algum tipo
intracked -> antes do comit
tracked -> pronto para o comit
